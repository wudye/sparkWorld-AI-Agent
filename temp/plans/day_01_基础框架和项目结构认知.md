### 任务1.4：本地启动和初步探索（1.5小时）

**操作步骤**：
```bash
1. cd /path/to/deer-flow
2. make check                           # 验证工具链
3. make install                         # 安装依赖
4. make dev                             # 启动所有服务
5. 浏览器打开 http://localhost:2026
6. 查看logs/下的4个日志文件
```

**关键观察**：
- 启动顺序（哪个服务先启）
- 日志中的关键信息（config加载、模型初始化）
