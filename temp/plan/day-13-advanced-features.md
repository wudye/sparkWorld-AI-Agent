## 📋 学习概念

### 1. 技能库架构

**设计**:
- 动态加载技能模块
- 每个技能是独立的 Python 包
- 技能定义包含元数据（名称、描述、参数）
- 与 LangChain Tools 集成

### 2. Model Context Protocol (MCP)

**作用**:
- 标准化工具集成方式
- 支持外部工具调用
- 异步流式支持

### 3. 代码沙箱

**用途**:
- 安全执行用户代码
- 资源隔离
- 文件系统虚拟化

### 4. 长期记忆系统

**组件**:
- 向量数据库存储
- 相似度检索
- 记忆摘要
